# 🌌 Coolest 404 Page

A visually stunning, interactive, and beautifully animated 404 Error page built using **Next.js** and **React**.

## 🚀 Quick Start

Run the project locally:

```bash
# 1. Install dependencies
npm install

# 2. Start the dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see it live!
